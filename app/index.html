IntroducingHQOCR

Our latest addition to the lineup of services we offer

HQOCR has many in-built features to meet the demands of today's enterprise solution. Please contact us for more information.

HQOCR is a neural net based optical character recognition (OCR) engine that recognizes line patterns.

HQOCR is an OCR engine that has the ability to recognize 100 languages out of the box. It is built with Deep Learning (DL) technology to train additional languages without code changes.

  • REST API Available
  • Asynchronous Batch and On-Demand modes
  • OCR selected regions
  • Support multiple output formats - HTML, PDF, TSV, XML
  • Train to support more languages and character patterns
  • Supports all Linux variants
  • and more...
